本說明提供取得 **2025 年度研發投資抵減完整計畫書** 與 **完整附件與報告** 的幾種方式，供列印、匯出 PDF 或送件使用。

---

| **完整計畫書** | 計畫書十章 + 附錄（重點摘要／基本資料與申請年度數據） | [plans/2025/2025_年度研發投資抵減_完整計畫書.md](./plans/2025/2025_年度研發投資抵減_完整計畫書.md) |

## 二、生成 PDF 或列印的作法

### 方式 A：用瀏覽器列印為 PDF（最簡單）

1. 若專案有使用 **MkDocs** 或其它 doc 站台：在瀏覽器開啟「研發投資抵減」專區，點到「2025 年度研發投資抵減 完整計畫書」頁面。
2. 按 **Ctrl + P**（或 檔案 → 列印），選擇「**另存為 PDF**」或「**Microsoft Print to PDF**」。
3. 儲存即得到一份 PDF 版完整計畫書。

若沒有建 doc 站台，可改用 **方式 B** 或 **方式 C**。

---

### 方式 B：用 Pandoc 轉成 PDF 或 Word

已安裝 [Pandoc](https://pandoc.org/) 時，在專案根目錄或 `docs` 目錄下執行：

**完整計畫書 轉成 PDF：**
```bash
pandoc "docs/rd-tax-credit/plans/2025/2025_年度研發投資抵減_完整計畫書.md" -o "2025_研發投資抵減_完整計畫書.pdf" --pdf-engine=xelatex -V CJKmainfont="Microsoft JhengHei"
```

**完整附件與報告 轉成 PDF：**
```bash
pandoc "docs/rd-tax-credit/plans/2025/2025_完整附件與報告.md" -o "2025_研發投資抵減_完整附件與報告.pdf" --pdf-engine=xelatex -V CJKmainfont="Microsoft JhengHei"
```

**研發工作報告完整版 轉成 PDF：**
```bash
pandoc "docs/rd-tax-credit/execution/work-reports/2025_研發工作報告_完整版.md" -o "2025_研發工作報告_完整版.pdf" --pdf-engine=xelatex -V CJKmainfont="Microsoft JhengHei"
```

**轉成 Word（.docx）：** 將上列指令中的 `-o "…pdf"` 改為 `-o "….docx"` 並移除 `--pdf-engine=…` 與 `-V CJKmainfont=…` 即可。

產出的 PDF 或 Word 即為可送件／存檔的完整計畫書或完整附件與報告。

---

### 方式 C：用 VS Code / Cursor 預覽後列印

1. 在 Cursor／VS Code 開啟 `docs/rd-tax-credit/plans/2025/2025_年度研發投資抵減_完整計畫書.md`。
2. 按 **Ctrl + Shift + V** 開啟 Markdown 預覽。
3. 在預覽視窗按右鍵 → **列印**，或 **Ctrl + P** 選擇「列印」／「另存為 PDF」。

---

## 三、送件前請再確認

- **統一編號**：完整計畫書中若仍為「＿＿＿＿＿＿＿＿」，送件前請改為公司統一編號。
- **附件與報告**：完整附件與報告合輯已含重點摘要（附件 I）、研究人員名冊、教育訓練與參訓名冊、成果報告摘要與效益、各季工作報告摘要、研發產出物清單；實際送件時依主管機關規定檢附，經費佐證等另見 [execution/](./execution/index.md)。
