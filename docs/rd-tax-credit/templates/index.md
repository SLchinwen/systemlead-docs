# 研發投資抵減：範本

本目錄存放與政府申請格式一致之範本，供每年撰寫計畫書與檢核表使用。

**產出計畫書格式（字體、標題項次）**：執行 `輸出DOCX.bat` 轉出計畫書時，會以「附件1-研發計畫重點摘要書(中小企業)」之 Word 格式為參考範本。可將政府提供之該 docx 複製至本目錄並命名為 `附件1-研發計畫重點摘要書(中小企業).docx`，即可在任意環境套用政府格式產出。
