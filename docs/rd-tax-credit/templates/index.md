# 研發投資抵減：範本

本目錄存放與政府申請格式一致之範本，供每年撰寫計畫書與檢核表使用。

**產出計畫書格式（字體、標題、縮排）**：執行 `輸出DOCX.bat` 轉出計畫書時，會以本目錄中的 **`ref.docx`** 為參考範本。

- **直接用原檔當範本**：將政府「附件1-研發計畫重點摘要書(中小企業).docx」複製到本目錄並命名為 **ref.docx** 即可，不需先做 Word 設定。可選用 [Setup-RefTemplate.ps1](../Setup-RefTemplate.ps1) 一鍵複製。
- **若轉出後表格沒有框線**：再依 [範本設定說明](./範本設定說明.md) 用 Word 開啟 ref.docx 設定表格樣式後儲存。
