# 研發投資抵減：範本

本目錄存放與政府申請格式一致之範本，供每年撰寫計畫書與檢核表使用。

**產出計畫書格式（字體、標題項次）**：執行 `輸出DOCX.bat` 轉出計畫書時，會以本目錄中的 **`ref.docx`** 為參考範本。請將政府提供之「附件1-研發計畫重點摘要書(中小企業).docx」**複製到本目錄並重新命名為 `ref.docx`**，即可套用政府格式產出（使用英文檔名可避免 CMD 括號與編碼問題）。
