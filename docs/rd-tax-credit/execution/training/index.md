# 研發相關教育訓練（執行佐證）

本目錄存放與研發投資抵減計畫相關之**教育訓練計畫、明細及參訓人員名冊**，作為申報與查核備查之用。訓練規劃依**每季任務所需技能**安排，並採**建議時數與學習目標**撰寫，產出格式對應 **TTQS（人才發展品質管理系統）** 常見要求。

## 與計畫書之對應

- 支援 **計畫書第六章**（研發內容與技術方法）所需之 AI 應用與資料治理能力。
- 佐證 **研發人員投入** 具備與計畫相關之學習與能力建構歷程。
