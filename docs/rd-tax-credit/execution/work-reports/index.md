# 研發工作報告紀錄（執行佐證）

本目錄存放各年度、各季之研發工作報告紀錄，作為研發投資抵減申報與**審計／查核備查**之用。每季報告記載研發目標、工作內容、人員參與、會議紀錄與產出，並保留簽名欄位供查核確認。

---

## 與計畫書之對應

- 對應 **計畫書第六章**（研發內容與技術方法）之執行過程紀錄。
- 佐證 **計畫書第九章**（研發成果與效益）之季度產出與演進。

## 備查說明

本目錄之工作報告及其附件（會議資料、分析表、系統設定紀錄等）均作為研發活動之「工作過程與人員參與紀錄」佐證，供研發投資抵減申報及主管機關查核備查使用。每月工作紀錄中註明之**建議的其他佐證**均為不涉及商業機密之類型，實際佐證存放位置與格式請依公司規定辦理。
