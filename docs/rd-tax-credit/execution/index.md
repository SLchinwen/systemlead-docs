# 研發投資抵減：執行佐證

本目錄存放研發計畫執行期間之佐證資料，對應計畫書第七章（人力）、第八章（經費）、第九章（成果）。

---

## 子目錄說明

| 子目錄 | 對應計畫書章節 | 建議內容 |
|--------|----------------|----------|
| [work-reports/](./work-reports/) | 六、研發內容與技術方法；九、研發成果與效益 | 各季研發工作報告紀錄（供審計／查核備查） |
| [training/](./training/) | 六、研發內容與技術方法；人力能力建構 | 教育訓練計畫、明細及參訓人員名冊（TTQS 格式） |
| [manpower/](./manpower/) | 七、研發人力配置與投入比例 | 工時紀錄、投入比例、人員名單與職務對應 |
| [expenditure/](./expenditure/) | 八、研發經費概況 | 經費分攤表、憑證對應、可認列項目說明 |
| [deliverables/](./deliverables/) | 九、研發成果與效益 | 報告、產出物、規格或程式之摘要或索引 |
