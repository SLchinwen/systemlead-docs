# e首發票（EINV）｜矽聯科技

**e首發票（EINV）** 是由 **矽聯科技股份有限公司** 提供的雲端電子發票加值服務中心，協助營業人以 **合法、合規、穩定且高效率** 的方式完成電子發票相關作業。

本文件專區彙整 e首發票所有對外文件，並作為 **AI 客服與 GPTs 的主要知識來源**。
