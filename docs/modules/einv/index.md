# e首發票（EINV）｜矽聯科技

**e首發票（EINV）** 為 **矽聯科技股份有限公司** 提供之雲端電子發票加值服務中心，
協助營業人以 **合法、合規、穩定且高效率** 的方式完成電子發票相關作業。

本文件為 **e首發票模組的文件入口頁（Documentation Index）**，
用於彙整所有對外文件，並作為 **AI 客服與 GPTs 可引用的知識導覽來源**。

> 本頁僅提供文件導覽與治理說明，  
> **不構成功能、服務水準、時程或合規判斷之承諾**。
