# B2B 電子發票下載操作手冊

## 一、手冊目的

本手冊說明 **e首發票（EINV）系統中「B2B 電子發票證明聯下載」的實際操作方式**，協助營業人下載已開立之 B2B 電子發票證明聯，以供列印、留存或提供給公司客戶使用。

> 本文件為 **非 SSOT 操作說明文件**，僅說明系統操作方式。  
> 發票性質與使用限制，請以相關規則與法規為準。

---

## 二、適用範圍說明

- 本功能 **僅適用於 B2B 電子發票**
- B2C 發票下載內容為「開立證明」，不可兌獎，僅供留存

---

## 三、功能說明

B2B 電子發票下載功能提供以下能力：

- 支援 **單筆或多筆（批次）下載**
- 可依下列條件查詢：
  - 發票日期區間
  - 發票號碼
  - 買受人統一編號
- 提供 **PDF 格式** 下載與列印

---

## 四、操作路徑

使用者可於以下任一路徑進行下載：

```
電子發票管理系統
＞ 銷項發票管理
＞ 待上傳發票管理
```

或

```
電子發票管理系統
＞ 銷項發票管理
＞ 已上傳發票管理
```

---

## 五、操作步驟

### （一）查詢發票

1. 進入【待上傳發票管理】或【已上傳發票管理】
2. 選擇發票日期區間
3. 視需要輸入發票號碼或統一編號

---

### （二）選擇下載格式

1. 於列印格式選項中，選擇 **【B（A4 紙）】**
2. 勾選欲下載之發票（可單筆或多筆）

---

### （三）下載或列印

1. 點選【列印】
2. 系統產生 PDF 檔案
3. 可直接下載或進行列印

---

## 六、注意事項

- B2B 發票 **無下載次數與時間限制**
- 請確認下載之 PDF 檔案內容正確後再提供給客戶
- 建議使用 **Chrome 或 Edge 瀏覽器** 操作
- 批次下載時，建議依月份分段查詢，避免檔案過大

---

## 七、常見問題（FAQ）

### Q：為什麼按下列印後沒有反應？
A：請確認列印格式是否選擇【B（A4 紙）】，並檢查瀏覽器是否允許下載 PDF；另可能是防毒軟體阻擋下載，可暫時關閉或加入例外後重試。

### Q：可以一次下載所有月份的發票嗎？
A：系統僅支援區間查詢，建議分月份下載以避免檔案過大。

### Q：下載的 PDF 可以直接寄給客戶嗎？
A：可以，請務必確認內容正確且未洩漏其他發票資訊。

### Q：下載的 PDF 檔案損毀或顯示不完整怎麼辦？
A：建議重新下載；若問題持續，請與 e首發票系統客服人員聯繫。

### Q：不同瀏覽器下載異常怎麼辦？
A：建議改用 Chrome 或 Edge，並確認瀏覽器版本為最新；若問題持續，請與 e首發票系統客服人員聯繫。

---

## 八、錯誤情境與處置方式

| 錯誤情境 | 建議處置 |
| --- | --- |
| PDF 無法開啟 | 重新下載；若仍異常，請與 e首發票系統客服人員聯繫 |
| 查詢不到發票 | 確認日期區間是否正確，或檢查發票是否已上傳至財稅平台 |
| 特定瀏覽器無法下載 | 請改用 Chrome 或 Edge 進行操作，並確認瀏覽器版本已更新 |
