# 指定日期開立電子發票操作手冊

## 一、手冊目的

本手冊說明 **e首發票（EINV）後台「指定日期開立電子發票」功能的實際操作方式**，協助營業人於符合法規與系統治理前提下，依需求指定發票開立日期完成發票開立作業。

> 本文件為 **非 SSOT 操作說明文件**，僅說明系統操作方式。  
> 發票取號責任、切本規則與時效限制，請以相關規則文件為準。

---

## 二、適用情境

- 跨期或跨月結帳
- 專案或合約約定指定開立日期
- 需延後開立發票之交易情境

---

## 三、功能說明

指定日期開立功能具備以下特性：

- 支援 **Excel 批次匯入** 指定日期訂單資料
- 系統自動檢核：
  - 發票組別
  - 字軌號碼
- 提供：
  - 查詢
  - 確認
  - 拋轉至財稅平台
- **指定開立日期須於規定天數內**（逾期將無法使用本功能）

---

## 四、操作步驟

### （一）發票字軌號碼與切本設定

在使用指定日期開立前，請先確認已完成對應日期的字軌與切本設定。

**操作路徑：**

```
發票訂單管理系統
＞ 媒體申報管理
＞ 發票字軌號碼管理（切本取號用）
```

操作說明：
1. 確認發票期別後點選【查詢】
2. 點選【新增切本配號】
3. 選擇批號（發票字軌）
4. 選擇切號用途（POS 機 P0～P35）
5. 填寫切本範圍（例：1-1、2-3）
6. 點選【確認送出】完成設定

---

### （二）指定日期開立發票

**操作路徑：**

```
發票訂單管理系統
＞ 銷項發票管理
＞ 訂單匯入開立（自訂日期）
```

操作流程：
1. 下載指定日期開立 Excel 範本
2. 填寫訂單資料、發票組別與指定開立日期
3. 點選【匯入 Excel 開立】並上傳檔案
4. 系統檢核完成後，點選【確認拋轉】完成開立

---

### （三）Excel 欄位說明（節錄）

| 欄位名稱 | 說明 | 必填 |
|---|---|---|
| 訂單編號 | 訂單唯一識別碼 | 是 |
| 發票組別 | 對應切本設定 | 是 |
| 發票日期 | 指定開立日期 | 是 |
| 買受人名稱 / 統編 | B2C / B2B 資訊 | 視情境 |
| 商品名稱 | 發票明細 | 是 |
| 數量 / 單價 / 金額 | 金額資訊 | 是 |
| 載具類型 / 編號 | 歸戶資訊 | 否 |

---

## 五、查詢指定日期發票

**操作路徑：**

```
發票訂單管理系統
＞ 待上傳發票管理
```

- 以發票日期查詢
- 系統會以標示方式顯示「指定日期開立」之發票資料

---

## 六、注意事項

- 指定開立日期 **必須於系統允許天數內**
- 匯入檔案中 **發票組別必填**
- 開立前須確認對應日期已完成切本設定
- Excel 範本不可修改標頭或加入公式

> 若因誤用造成逾期、重號或異常情形，系統可能限制使用本功能。

---

## 七、常見問題（FAQ）

### Q：若指定日期超過限制天數會如何？
A：系統將拒絕使用指定日期功能，請改用當日日期或其他方式處理。

### Q：未填發票組別會發生什麼事？
A：系統會提示序時或字軌異常，無法完成匯入。

### Q：開立後多久會上傳至財稅平台？
A：系統依排程自動上傳，亦可視需求手動上傳。

---

## 八、錯誤情境與處置方式

| 情境 | 建議處置 |
|---|---|
| 發票號碼重複或跳號 | 作廢後重新開立 |
| 指定日期不符規範 | 調整日期後重新匯入 |
| Excel 格式錯誤 | 修正後重新上傳 |
| 載具格式錯誤 | 修正載具資訊 |
