# EasyStore 手動作廢電子發票操作手冊

## 一、手冊目的

本手冊說明營業人於 **EasyStore 平台** 上 **手動觸發作廢電子發票** 的完整操作步驟，適用於發票資訊錯誤需立即作廢並重開，或訂單取消、金額異常需依規定作廢發票時。

> 本文件為 **非 SSOT 操作說明文件**，僅說明系統操作方式。

---

## 二、功能說明與禁止使用情境

**適用情境**：
- 發票資訊錯誤需立即作廢並重開時
- 訂單取消或金額異常，需依規定作廢發票

**禁止使用情境**：
- 發票已跨期，不得直接作廢，需以折讓單或報備國稅局處理
- 非原發票開立人，不得作廢該發票
- 不得對已使用或兌獎之發票進行作廢

---

## 三、操作步驟

### （一）進入訂單明細

- 登入 EasyStore 後台
- 路徑：**訂單管理 > 所有訂單 > 點選訂單編號 > 訂單明細 > e首發票電子發票系統**

### （二）選擇需作廢的發票

- 於電子發票區塊，找到需作廢的發票號碼
- 點擊【作廢發票】按鈕

### （三）填寫作廢原因

- 系統將提示輸入作廢原因，請填寫如：訂單取消、發票金額錯誤、統編填寫錯誤等
- 點擊【確認送出】

### （四）確認作廢結果

- 作廢完成後，系統會回傳「作廢成功」訊息
- 可於「電子發票」頁面查詢作廢狀態，並確認是否回傳財政部平台
- **作廢僅在財政部平台回傳「成功」後才算正式生效**，請務必查詢確認

---

## 四、注意事項

- 作廢僅能於 **當期** 進行，跨期需依國稅局規定以折讓處理
- 作廢原因需清楚填寫，避免後續查核爭議
- 發票作廢後，若需補開，請重新開立新發票
- 建議於每日結束前，至 e首發票後台確認作廢回傳狀態

---

## 五、常見問題（FAQ）

| 問題 | 解答 |
| --- | --- |
| 為什麼發票無法作廢？ | 請確認是否為跨期或已兌獎發票，跨期需折讓，已兌獎不可作廢 |
| 作廢後還能查到發票嗎？ | 可以，發票會顯示作廢狀態，並保留於系統中以供稽核 |
| 作廢後要如何補開？ | 請依原訂單資訊重新手動開立新發票 |
| 作廢申請送出後多久能查詢結果？ | 一般需 5~10 分鐘，請至 e首發票後台確認，或隔日再次查詢 |
| 如果誤刪作廢怎麼辦？ | 誤刪後無法復原，請重新開立正確發票，並保留異常紀錄以備查核 |
| 遇到錯誤代碼 401/500？ | 檢查 SALT 與統編授權是否正確，若持續異常請聯繫客服 |

---

## 六、錯誤情境與處置方式

| 錯誤情境 | 處置方式 |
| --- | --- |
| 發票跨期 | 不可直接作廢，需改以折讓單處理並報備國稅局 |
| 發票號碼無法回傳 | 確認是否有網路異常，或登入 e首發票後台檢查作廢紀錄 |
| 遇到系統錯誤 401/500 | 檢查 SALT 與統編授權，若仍異常請聯繫客服 |
| 發票已兌獎 | 不得作廢，需保留紀錄並報備國稅局 |
| 誤刪作廢 | 無法復原，請重新開立正確發票，並保留紀錄以備查核 |

---

## 七、合規提醒

- **作廢時限**：僅能於當期作廢，跨期需折讓
- **禁止事項**：不得作廢已兌獎之發票；不得跨期作廢，應依規定開立折讓單
- **保存要求**：作廢發票與相關資料需保存至少 5 年，備稅務查核
