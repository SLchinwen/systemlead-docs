# e首發票｜SOP（Standard Operating Procedure）

本區彙整 **e首發票常見關鍵情境的判斷 SOP**，  
用於協助使用者與客服人員，在「不知道該用哪個功能」時，快速做出正確決策。

> SOP 目的在於「**怎麼判斷**」，  
> 實際操作請一律導向對應的操作手冊（Manual）。
